# PlayMarketReviews
Python 2.7 required

+ Check Chrome Version: Open in new tab: ```chrome://settings/help```
+ Download [ChromeWebDriver](http://chromedriver.chromium.org/downloads) for your Chrome version and put to this folder
+ ```pip install selenium```
